# CC电脑播放网易云音乐

## 使用说明：

* 确保CC电脑安装了扬声器/手持电脑为喧闹电脑

## 使用

* wget代码到mc中的cc电脑

*  `play [id/lid/dfpwm] [id] {once/cycle}`
    * `id/lid/dfpwm`: 选择id类型 id=**音乐id** lid=**歌单id*** dfpwm=**电脑内dfpwm播放**
    * `id`: 音乐或者歌单id | 当为dfpwm播放时为文件路径
    * `once/cycle`: 播放模式 once**单次播放** cycle**循环播放***(默认单次播放)
